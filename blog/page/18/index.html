Python中print中的逗号

3Jul

以往可能会写这样的代码：

print x 
print y

 当然，x、y在前面都是定义过的变量，此时，如果在x后面加一个逗号，再看输出结果，print就不会再换行了，只会在输出完x后加一个空格接着就输出y了，这点不错！

针对github的两条设置

21Jun

刚刚看到github的help页面上有两条关于git的设置，是针对github用的：

git config --global github.user USERNAME 
git config --global github.token TOKEN

 其中的TOKEN要从github网站的帐户管理中得到，而且这个TOKEN和密码相关，如果你更改了密码，需要重新设置。但我还不清楚设置了这个对于github的提交有什么特殊的好处，等以后用用再说吧。

M555b鼠标在Windows7下的使用

18Jun

为了减少一个USB接口的占用买了个蓝牙鼠标，当然这是因为我的电脑（mbp13）只有两个USB。但M555b这款鼠标在Mac系统下工作非常好，不过到了windows7下就异常了，根本连不上，刚刚在百度知道中看到一个方法可用，分享一下！

1.在mac系统下删除m555b这个设备。

2.到widnows7下用系统的蓝牙服务扫描到鼠标，然后用触摸板右键点扫描到的鼠标，选属性，在蓝牙服务中勾选“鼠标、键盘驱动HID”这个选项。然后你会发现已经匹配完成了，此时鼠标就能用了。

3.再回到mac下重新加上m555b这个设备就可以了：）
