Python中一个bt的列表方法

Jun 5th, 2011

如果你感觉列表这种数据结构就已经很bt了的话，再看看这个吧，我感觉这简直就在破坏程序员的正常思维！

example_list = [2,3,4]

other_list = [i*2 for i in example_list if i > 2]

执行之后other_list就是[6,8]了！！！

意思就是说，对于example_list中的元素进行遍历，将大于2的元素乘上2构成新的列表！

这样的自由构建方式，是被推广使用的么？！
