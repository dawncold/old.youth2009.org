eclipse安装EGit插件

Feb 23rd, 2011 

        <p>其实根本没必要写成文章的，eclipse的插件安装不外乎：打开Install Software的界面，输入某个插件的地址，然后一路next</p>

EGit的官方下载页面是：http://www.eclipse.org/egit/download/

官方推荐的用于eclipse安装地址是：http://download.eclipse.org/egit/updates

所以我们就在eclipse的Help-》Install New Software的那个地址中输入：http://download.eclipse.org/egit/updates

随后能够得到2个安装条目，我倒是没怎么了解详情，全部安装了，next后就是下载相关的文件自动安装。

安装完毕后重启eclipse，在已经建立的项目上右键单击，选择Team，Share Project，再选GIT方式share。下一步就是建立针对某个项目的git仓库，直接点击Create xxx 按钮即可，这样就在本地完成了git管理相关项目的步骤。人比较懒，不想放出图来：）
